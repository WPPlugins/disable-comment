== Description ==

Disable Comments - Allow Admin to Disable comments, trackbacks and/or pingbacks globally or for specific post types, categories, tags, referrers and much more.


== Installation ==

1. Download plugin zip file to your computer and extract in a folder
1. Upload `wsx-disable-comments` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
